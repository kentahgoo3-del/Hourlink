8. Automation rules
This part is critical.

Automation engine should support:
when task created → notify freelancer

when quote sent → notify company

when quote approved → create project/work order

when timer starts → task status becomes in progress

when timer stops → add billable entry

when task marked complete → prepare invoice draft

when invoice overdue → send reminder

when payment confirmed → mark invoice paid

when no response to quote in X days → send reminder

when due date approaches → send task reminder

when task receives company comment → alert freelancer

when manual entry added → recalculate totals

when quote revised → maintain revision history

when company uploads file → notify freelancer
